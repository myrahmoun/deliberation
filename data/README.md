# Data

## Remesh verbatim maps (`anticlustering/`)

Three Remesh deliberation datasets used for anti-clustering experiments:

- `campus_protests_verbatim_map.csv`
- `foreign_intervention_verbatim_map.csv`
- `right_to_assemble_verbatim_map.csv`

Source: https://github.com/akonya/polarized-issues-data

## Model weights

LoRA adapters for BGE-large and sentence-T5-XL live in the `embeddings-for-preferences/` submodule at the repo root. Run `git submodule update --init` to fetch them.
